## How to Play

1. Open `index.html` in a modern web browser
2. Click "START GAME"
3. Use WASD to move, mouse to look around, click to shoot
4. Survive the tacky holiday apocalypse!

## Controls

- **W/A/S/D** - Move forward/left/backward/right
- **Mouse** - Look around
- **Click** - Shoot
- **Q/E** or **Arrow Keys** - Turn left/right (alternative)
